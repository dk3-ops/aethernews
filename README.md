## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- Docker & Docker Compose
- Git
- Kubernetes CLI (kubectl)

### Installation

```bash
# Clone repository
git clone https://github.com/dk3-ops/aethernews.git
cd aethernews

# Install dependencies
npm run install:all

# Setup environment
cp .env.example .env

# Start development environment
docker-compose up -d

# Run migrations
npm run migrate

# Start development server
npm run dev
```
